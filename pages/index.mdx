# TON Dev

**tondev.org** is a community-lead collection of documentation about development for TON blockchain.

[TON](https://ton.org) is a fully decentralized layer-1 blockchain designed by [Telegram](https://telegram.org) to onboard billions of users. 
It boasts ultra-fast transactions, tiny fees, easy-to-use apps, and is environmentally friendly.
